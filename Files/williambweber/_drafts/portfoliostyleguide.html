Portfolio Style Guide

I will list the specifics of what I have done to affect the css style on my page.

External Styling Sheet

  • I have an external css file that is linked to each index page from the styles folder.
  • The background is multi-linear and goes from gold to purple (Go Dawgs!!!).
  • My header is black, arial, size 24px, font weight 400, text color white, aligned left, and padded 12px.
  • My Navigation - I created a navbar floated to the left, padded t/b 14px & r/l 16px, font is arial, size 16px, color white, text aligned center, and background color is black.
  • Main has a font size of 1.2em and a padding of 1em.
  • Main Paragraph has a padding of.4em and a text indent of 35px.
  • Main h1 has a padding of .4em, font is arial, font size is 24pt, and font weight bold.
  • Main h2 has a padding of .5em, font is arial, font size is 22pt, font style is italic, and font weight is bold.
  • Main li has a text indent of .5em.
  • All ul have a class id of "bullets". List style type is inherit. List style position is inside. Padding left is 4em. padding top is .5em.
  • Footer is position fixed, bottom 0, width 100%, background is black, and border is black and solid. Font color is white, arial, size 14px, and text aligned center.

External Layout Sheet

  • Header h1 is text aligned center.
  • Navbar padding is 0 on the top/bottom, 12% on the right, and 16% on the left.
  • Navbar list anchor is width 15%.
  • .center is position absolute, 10% on the top, and 43% on the left.
  • .content is position inherit, width 50%, and padding is top/bottom 0 and right/left is 25%.
  • .content h2 is position inherit and text align center.
  • .index is position inherit, padding is 1% for top/bottom and 30% for left/right, width is 40%.
  • .links is position absolute and padding is 1% for top/bottom and 35% for left/right.
  • .labs is floated left, width is 300 pixels, and padding is 1% on top/bottom and 40% for left/right.
  • .index h2 is text aligned center.
  • .stylingguide is position inherit, width is 40%, and padding is (1% top, 30% right, 2% bottom, and 30% left).

Internal Style

  • No internal styling on any of my pages.
